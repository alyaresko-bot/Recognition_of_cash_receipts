# Recognition_of_cash_receipts
Telegram-бот для автоматизации учетных записей: принимает фото, извлекает их через данные QR (приоритет) или OCR, сохраняет дубликаты и сохраняет структуру в таблице. Поддерживает мультитенантность, лимиты и биллинг, очередность задач и мониторинг — готов к использованию как SaaS.
